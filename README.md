### 安装方法

1 克隆下载

```bash
# github
git clone https://github.com/guapit/guapit-typora-theme.git
# gitee
git clone https://gitee.com/guapit/guapit-typora-theme.git
```

2 打开Typora文档软件

`文件(F)` > `偏好设置` > 选择 `外观` > `打开主题文件夹`

将下载好的 `theme` 目录的所有文件复制到 `Typora\themes`即可
